JHU Software Concepts Module 4 Documentation
============================================

Overview
--------

This project is a Flask and PostgreSQL application for analyzing
GradCafe graduate admissions data.

The application supports:

* Loading applicant data into PostgreSQL
* Running analytics queries
* Updating the database with new records
* Displaying results through a Flask web interface
* Automated testing with Pytest and GitHub Actions
