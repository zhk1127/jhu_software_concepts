Module 1 Personal Website

Author:
Hongkang Zhang

Description:
This project is a personal website built using
Python Flask, HTML, and CSS.

Features:
- Homepage with biography and profile image
- Projects page
- Contact page
- Navigation bar with active page highlighting

Requirements:
- Python 3.10+
- Flask

Instructions to Run:

1. Open terminal in the module_1 folder

2. Install requirements:
pip install -r requirements.txt

3. Run the Flask application:
python run.py

4. Open browser:
http://127.0.0.1:8080
